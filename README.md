	1. 口罩下方的数字序列是否清晰：（可取值 “是”和“否”）
	2. 字体是否是斜纹45度喷墨上去的：（可取值 “是”和“否”）
	3. 金属条是否光滑：（可取值 “是”和“否”）

	5. 口罩上的中英文是否清晰：（可取值 “是”和“否”）
	6. 是否有QS认证或LA认证：（可取值 “是”和“否”）

* 训练样例集D：目标函数的正例和反例

求解：
* H中的一假设h， 使对于X中任意x , h(x) = c(x)
